# 11-Stage Sacred AI Engine Workflow

This workflow is designed to be run locally and controlled from your device through a CLI + Make targets.

## Stage 1 — Source Ingestion & Normalization
- Gather source briefs, prior chat exports, and brand directives.
- Convert to structured markdown and JSON for traceability.
- Output: `focus_ai/docs/source_digest.md`.

## Stage 2 — Brand Voice Calibration
- Lock tone: bold, spiritual-modern, practical, sales-aware.
- Define do/don't lists for claims, style, and compliance.
- Output: `focus_ai/docs/brand_voice.md`.

## Stage 3 — Offer Architecture
- Build service ladders, value stack, and CTA map.
- Output: `focus_ai/docs/offers.md`.

## Stage 4 — Visual Direction System
- Create sacred geometry visual rules, typography hierarchy, and motion language.
- Output: `focus_ai/docs/visual_system.md`.

## Stage 5 — Conversion Copy Engine
- Generate high-impact homepage, landing sections, and CTA variants.
- Output: `focus_ai/site/homepage_copy.md`.

## Stage 6 — Trust & Compliance Layer
- Add legal-friendly language and IP documentation checklist.
- Output: `focus_ai/docs/ip_and_compliance.md`.

## Stage 7 — Content Expansion
- Produce long-form educational content and lead magnets.
- Output: ebooks in `focus_ai/ebooks/`.

## Stage 8 — Web Change Operations
- Apply content updates (including phone number normalization to `2172576222`).
- Output: update logs in `focus_ai/site/change_log.md`.

## Stage 9 — QA & Professionalism Audit
- Proofreading, consistency pass, grammar pass, readability pass.
- Output: `focus_ai/docs/qa_report.md`.

## Stage 10 — Launch Readiness
- Deployment checklist, rollback checklist, analytics verification.
- Output: `focus_ai/docs/launch_checklist.md`.

## Stage 11 — Continuous Optimization Loop
- Weekly KPI review and prompt refinement loop.
- Output: `focus_ai/docs/optimization_loop.md`.
