# Vendor / Net Terms Setup Checklist

This is a setup checklist only. It is not a completed submission log.

## Data To Gather Before Applying

1. Legal entity name for each company
2. EIN letter
3. Secretary of state registration proof
4. D-U-N-S number if required
5. Business address and phone
6. Authorized signer information
7. Bank reference and trade references if requested
8. Separate secure password storage for each vendor account

## Security Note

Do not reuse one shared password for every vendor account. Use unique passwords and store them in a password manager or other approved secret-storage system.

## Suggested Tracking Fields

- Vendor
- Company entity used
- Date started
- Date submitted
- Login email
- Terms requested
- Current status
- Credit limit approved
- Documents uploaded
- Follow-up date
