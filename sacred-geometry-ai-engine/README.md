# Sacred Geometry AI Engine

An autonomous AI-powered system for researching sacred geometry architecture, designing construction methods, generating long-form educational books, and automating publishing + monetization workflows.

## Vision

Create a scalable ecosystem that combines:

- Architecture innovation
- Spiritual alignment education
- Automated digital publishing
- Passive income systems

## Core Functions

1. Sacred geometry architectural research
2. Verified source extraction and claim scoring
3. Book outline and chapter generation
4. Geometry concept and design iteration
5. Construction system optimization
6. Compliance and safety review templates
7. Frequency/chakra alignment documentation
8. Marketing automation and product distribution

## Automation Pipelines

### 1) `automation/update_engine.py`
- Appends timestamped update checks to all core markdown files
- Generates reusable section scaffolds for research, claims, chapters, and monetization
- Tracks UTC timestamps for auditability

### 2) `automation/ai_generator.py`
- Generates new manuscript chapter blocks
- Generates architecture/geometry concept blocks
- Exports markdown updates for downstream publishing workflows

### 3) `.github/workflows/ai_engine.yml`
- Runs daily on cron
- Executes the AI generator
- Commits and pushes generated updates automatically when changes exist
