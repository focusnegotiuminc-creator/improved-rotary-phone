# AGENTS Instructions - Sacred Geometry AI Engine

## Objective
Maintain this repository as an autonomous content and architecture generation system with clear, auditable markdown outputs.

## Authoring Rules
1. Keep markdown files structured with clear sections, templates, and timestamps.
2. Use UTC timestamps in automation outputs.
3. Preserve reproducibility: deterministic headings + labeled generated blocks.
4. Avoid destructive overwrites unless explicitly requested.

## Python Rules
1. Python scripts should be standard-library only unless dependency additions are approved.
2. Keep generators modular (`generate_*` helpers + `update_files()` orchestration).
3. Write markdown in append mode for iterative growth, except explicit export scripts.

## Workflow Rules
1. GitHub Actions should skip empty commits.
2. Automation commits must identify bot user/email.
3. Include meaningful commit messages with timestamps.

## Product Focus
Optimize for:
- Automated book writing

- Passive income packaging and distribution
